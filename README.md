# 🌙 Luna (ลูน่า) - AI Personal Assistant & Companion

Luna เป็นแอปพลิเคชัน Chatbot Assistant ที่ถูกออกแบบมาให้เป็นมากกว่า AI ทั่วไป โดยเน้นไปที่การเป็น **"เพื่อนคุยแก้เหงา"** และ **"พื้นที่ปลอดภัย (Safe Zone)"** สำหรับผู้ใช้ มีระบบทำความเข้าใจอารมณ์ บทสนทนาที่หลากหลาย และเรียนรู้ตัวตนของผู้ใช้ได้

## 🌟 ฟีเจอร์หลัก (Key Features)

- **🧠 Deep Memory (ระบบจำตัวตน):** ลูน่าสามารถเรียนรู้และจดจำสิ่งที่ผู้ใช้บอก เช่น ชื่อ, ความชอบ, สิ่งที่ไม่ชอบ, และเป้าหมายส่วนตัว เพื่อนำมาใช้ในบทสนทนาต่อไป
- **📝 Secretary Mode (โหมดเลขา):** เพียงพิมพ์คำว่า `"ลูน่า"` ระบบจะเข้าสู่โหมดเลขาเพื่อช่วยคุณจดโน้ตตารางงานหรือเตือนความจำต่างๆ (บันทึกและลบได้ตลอดเวลา)
- **❤️ Mood Tracking (บันทึกอารมณ์):** กดปุ่มหัวใจเพื่อบันทึกอารมณ์ในแต่ละวัน และสามารถเรียกดูประวัติอารมณ์ของคุณได้ตลอดเวลา
- **💬 Natural & Companion Chat:** รองรับคลังคำศัพท์วัยรุ่น, คำสแลง, คำพ้องความหมาย และหัวข้อพูดคุยในชีวิตประจำวัน (เช่น บ่นเรื่องงาน, ปวดหลัง, เรื่องเงิน, อกหัก) ตอบกลับด้วยความหลากหลายไม่จำเจ
- **🩹 Healing System:** ระบบตรวจจับคำหยาบหรืออารมณ์หงุดหงิด โดยลูน่าจะตอบกลับด้วยความใจเย็น สุภาพ และมีข้อความฮีลใจให้กำลังใจแทรกอยู่เสมอ
- **💾 Local Storage:** ข้อมูลทุกอย่าง ไม่ว่าจะเป็นประวัติแชท, อารมณ์, หรือโน้ต จะถูกบันทึกไว้ในเครื่องของคุณ ปลอดภัยและเป็นส่วนตัว

## 🛠️ เทคโนโลยีที่ใช้ (Tech Stack)
- React.js
- Vanilla CSS (Modern Glassmorphism Design & Micro-animations)
- Lucide React (Icons)
- LocalStorage API

## 🚀 การติดตั้งและใช้งาน (Installation)

1. Clone repository หรือดาวน์โหลดซอร์สโค้ด
2. ติดตั้ง Dependencies:
   ```bash
   npm install
   ```
3. รัน Development Server:
   ```bash
   npm run dev
   ```
4. เปิดเบราว์เซอร์ไปที่ `http://localhost:5173`

## 💡 วิธีการใช้งานและคำสั่งเบื้องต้น
- **เรียกใช้งานเลขา:** พิมพ์ `"ลูน่า"` 
- **เปลี่ยนชื่อ:** พิมพ์ `"เราชื่อ [ชื่อ]"` หรือ `"เรียกเราว่า [ชื่อ]"`
- **ดูสิ่งที่ลูน่าจำได้:** พิมพ์ `"จำอะไรได้บ้าง"`
- **บันทึกอารมณ์:** พิมพ์ `"บันทึกอารมณ์"` หรือกดปุ่ม 💖 ข้างช่องแชท
- **บ่นระบายอารมณ์:** พิมพ์เรื่องในชีวิตประจำวันได้เลย เช่น `"ปวดหลัง"`, `"ไม่มีเงิน"`, `"เบื่องาน"`, `"เหงาจัง"`
